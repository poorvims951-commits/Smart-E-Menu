 Smart E-Menu

Description
Smart E-Menu is a digital restaurant menu system that allows customers to view menu items electronically instead of using traditional paper menus. It helps restaurants update menu items easily, reduces physical contact, and improves the overall customer experience. The system supports quick access to food details and pricing in a simple and user-friendly way.

 Features
- Digital display of restaurant menu  
- Easy menu updates without reprinting  
- Improved hygiene and reduced paper usage  
- Simple and user-friendly interface  
- Fast access to food items and prices  


Technologies Used
- Node.js  
- JavaScript  
- HTML, CSS  
- JSON (for data storage)  
- Git & GitHub  

Installation & Setup

1. Clone the repository:
   git clone https://github.com/poorvims951-commits/Smart-E-Menu.git

Navigate to the project folder:

cd smart-e-menu

Install dependencies:

npm install

Start the server:

node server.js

Open your browser and visit:

http://localhost:3000
